## 🔧 Outils de Création Rapide

### Canva (Recommandé pour débutants)
1. Créer un compte gratuit
2. "Créer un design" → Dimensions personnalisées
3. Pour la bannière: 1544 x 500 px
4. Utiliser templates "Tech" ou "App"
5. Personnaliser avec vos couleurs
6. Télécharger en PNG

### Figma (Pour designers)
1. Nouveau fichier
2. Frame: 1544 x 500
3. Design avec composants
4. Export: PNG 2x

### Photoshop
1. Nouveau document: 1544 x 500 px, 72 DPI
2. Mode couleur: RVB
3. Design
4. Enregistrer pour le web: PNG-24

## ✅ Validation Avant Upload

### Checklist Technique
- [ ] Dimensions exactes respectées
- [ ] Format PNG (pas JPG pour les icônes)
- [ ] Taille fichier < 1 MB par image
- [ ] Pas de texte trop petit (lisible à 100%)
- [ ] Contraste suffisant
- [ ] Pas de contenu protégé par copyright
- [ ] Pas de marques tierces sans permission

### Checklist Qualité
- [ ] Images nettes (pas floues)
- [ ] Couleurs cohérentes avec le plugin
- [ ] Texte en anglais
- [ ] Design professionnel
- [ ] Représente bien les fonctionnalités
- [ ] Attire l'attention

## 📊 Optimisation des Images

### Compression Recommandée
- **TinyPNG** - https://tinypng.com (gratuit)
- **ImageOptim** - https://imageoptim.com (Mac)
- **Squoosh** - https://squoosh.app (web)

### Objectif
- Réduire la taille sans perte de qualité visible
- Banner: < 200 KB idéalement
- Icon: < 50 KB idéalement
- Screenshots: < 150 KB chacun
