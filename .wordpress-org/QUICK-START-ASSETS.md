# 🎨 Guide Rapide - Créer les Assets en 30 Minutes

## 🚀 Méthode Rapide avec Canva (Recommandé)

### Étape 1 : Créer un Compte Canva (Gratuit)
1. Aller sur https://www.canva.com
2. S'inscrire gratuitement
3. Choisir "Créer un design"

---

## 📐 Asset 1 : Bannière Principale (1544x500px)

### Sur Canva :
1. Cliquer "Créer un design"
2. Choisir "Dimensions personnalisées"
3. Entrer : **1544 x 500 pixels**
4. Cliquer "Créer un design"

### Design Recommandé :

**Fond :**
- Gradient : Du violet (#667eea) au violet foncé (#764ba2)
- Ou utiliser un template "Tech" ou "App"

**Texte à Ajouter :**
```
[GAUCHE]
📊 Ad Banner Scanner
Advanced Ad Manager for WordPress

[DROITE - Badges]
📊 Real-time Stats
🎯 Smart Targeting
💰 Revenue Tracking
```

**Éléments Visuels :**
- Icône de graphique ou statistiques
- Mockup de smartphone (optionnel)
- Couleurs : Violet (#667eea), Vert (#00D924), Blanc

### Télécharger :
1. Cliquer "Partager" → "Télécharger"
2. Format : **PNG**
3. Qualité : **Haute**
4. Nommer : `banner-1544x500.png`
5. Sauvegarder dans : `/Users/kameltalbi/Desktop/`

## 📸 Screenshots (7 images)

### Comment Prendre les Screenshots :

1. **Installer le plugin localement** (si pas déjà fait)
2. **Activer le plugin**
3. **Naviguer vers chaque page**
4. **Prendre des captures d'écran**

### Screenshots à Prendre :

**Screenshot 1 : Dashboard**
- Aller sur : AdWPTracker → Dashboard
- Capturer : Toute la page avec graphiques
- Nommer : `screenshot-1.png`

**Screenshot 2 : Liste des Annonces**
- Aller sur : AdWPTracker → Annonces
- Capturer : Liste avec shortcodes visibles
- Nommer : `screenshot-2.png`

**Screenshot 3 : Création d'Annonce**
- Aller sur : Nouvelle Annonce
- Capturer : Formulaire avec options sticky
- Nommer : `screenshot-3.png`

**Screenshot 4 : Configuration Zone**
- Aller sur : Nouvelle Zone
- Capturer : Options de zone
- Nommer : `screenshot-4.png`

**Screenshot 5 : Statistiques**
- Aller sur : AdWPTracker → Statistiques
- Capturer : Tableau des stats
- Nommer : `screenshot-5.png`

**Screenshot 6 : Sticky Mobile (Mockup)**
- Utiliser un outil de mockup : https://mockuphone.com
- Ou capturer depuis un vrai mobile
- Montrer le sticky footer en action
- Nommer : `screenshot-6.png`

**Screenshot 7 : Paramètres**
- Aller sur : AdWPTracker → Paramètres
- Capturer : Page des paramètres
- Nommer : `screenshot-7.png`

### Optimiser les Screenshots :
1. Aller sur https://tinypng.com
2. Uploader chaque screenshot
3. Télécharger les versions compressées

## 💡 Conseils

1. **Simplicité** : Designs épurés fonctionnent mieux
2. **Lisibilité** : Texte gros et clair
3. **Cohérence** : Mêmes couleurs partout
4. **Qualité** : Images nettes, pas floues
5. **Optimisation** : Compresser pour web
