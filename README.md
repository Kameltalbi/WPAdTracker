Professional WordPress advertising management plugin with **unique mobile sticky footer**, real-time statistics, and advanced targeting.

## 🌟 Key Features

### 📱 Mobile Sticky Footer (Unique!)
- Industry-first sticky footer ad for mobile & tablet
- Automatically hidden on desktop
- Professional close button
- Session-based dismissal

### 🎨 4 Ad Types
- **Image Ads** - Upload with media library
- **HTML/JavaScript** - Google AdSense support
- **Text Ads** - Native advertising
- **Video Ads** - YouTube, Vimeo, MP4

### 📊 Real-Time Statistics
- Live impression tracking
- Click-through rate (CTR)
- Daily/weekly/monthly reports
- CSV export

### 🎯 Advanced Targeting
- Device targeting (Mobile, Tablet, Desktop)
- Schedule campaigns (Start/End dates)
- Unlimited zones
- Position control

## 🚀 Quick Start

### Installation

1. Upload `adwptracker` folder to `/wp-content/plugins/`
2. Activate plugin in WordPress
3. Go to **AdWPtracker** menu
4. Create your first zone and ad

### Basic Usage

```php
// Display zone with ID 1
[adwptracker_zone id="1"]

// In PHP template
<?php echo do_shortcode('[adwptracker_zone id="1"]'); ?>
```
